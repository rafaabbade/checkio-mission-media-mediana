Enunciado

Escreva uma função que, dados dois números inteiros distintos A e B, calcule o menor inteiro possível C tal que a média e a mediana de A, B e C sejam iguais.

Entrada Dois números inteiros A e B.

Saída: Um número inteiro, representando o menor C possível que faz a média e a mediana de A, B e C iguais.

Example:

countConsecutiveSummers(42) == 4
countConsecutiveSummers(99) == 6
{% else %} 
    media_mediana(1, 2) >> 0

media_mediana(6, 10) >> 2

media_mediana(1, 1000000000) >> -999999998
